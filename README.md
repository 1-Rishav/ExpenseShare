## ğŸ“– Overview

ExpenseShare is a robust, full-stack web application designed to simplify the process of sharing expenses among groups. Inspired by Splitwise, it features a powerful **Min-Cash-Flow Algorithm** that minimizes the number of transactions required to settle debts within a group.

### Why ExpenseShare?

| Problem | Solution |
|---------|----------|
| Complex debt graphs between friends | Intelligent debt simplification algorithm |
| Manual calculation errors | Automatic split calculations (Equal, Exact, Percentage) |
| Confusing "who owes whom" | Clear, simplified balance view |
| Multiple unnecessary transactions | Optimized settlement plan |

### The Problem
